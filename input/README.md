# Restricted data instructions

The Swiss Active Labor Market Policy data are restricted-use data from SwissUbase/FORSbase.

To reproduce the project:

1. Request access at: https://www.swissubase.ch/en/researcher/my-downloads/113929/25900/download-details
2. Download `swissubase_1203_1_0.zip`.
3. Place the zip file in this `input/` folder, or unzip it and place `1203_ALMP_Data_E_v1.0.0.csv` in this `input/` folder.
4. Run `make` from the repository root.

The raw data are not committed to GitHub because access is restricted.
