# Data dictionary for variables used in replication

| Variable | Description | Units/source |
|---|---|---|
| `treatment6` | Program category: no program, job search, vocational, computer, language, employment, personality | Raw Swiss ALMP data |
| `canton_german` | Indicator for German-speaking canton | Raw Swiss ALMP data |
| `start_q2` | Indicator for program start in months 4-6 of unemployment spell | Raw Swiss ALMP data |
| `employed1`-`employed36` | Monthly employment indicators after unemployment registration | Raw Swiss ALMP data |
| `age` | Age of unemployed individual | Years |
| `female` | Female indicator | Binary |
| `foreigner_b`, `foreigner_c` | Foreigner permit-status indicators | Binary |
| `swiss` | Swiss citizenship indicator | Binary |
| `married` | Married indicator | Binary |
| `past_income` | Previous income | CHF |
| `employability` | Caseworker employability rating | Ordered: low, medium, high |
| `emp_share_last_2yrs` | Fraction of months employed in previous two years | Share |
| `emp_spells_5yrs` | Employment spells in previous five years | Count |
| `ue_spells_last_2yrs` | Unemployment spells in previous two years | Count |
| `unemp_rate` | Local unemployment rate | Percent/rate |
| `gdp_pc` | Regional GDP per capita | CHF |
| `city_big`, `city_medium`, `city_no` | City-size indicators | Binary |
| `canton_moth_tongue`, `other_mother_tongue` | Mother-tongue indicators | Binary |
| `cw_*` | Caseworker characteristics such as age, gender, education, tenure, cooperative style, and own unemployment experience | Raw Swiss ALMP data |
| `prev_job_*` | Previous job/sector/skill indicators | Raw Swiss ALMP data |
| `qual_*` | Qualification/education indicators | Raw Swiss ALMP data |
| `ue_cw_allocation1`-`ue_cw_allocation6` | Caseworker assignment-process indicators | Raw Swiss ALMP data |
